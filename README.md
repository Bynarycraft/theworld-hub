# The World Museum Demo

A Babylon.js browser demo that showcases a 3D hub globe and Nigeria festival missions for Igbo, Yoruba, and Hausa. The demo is a short, playable vertical slice designed for 3-7 minutes of play.

## Demo Flow
- Hub globe: rotate and select Africa.
- Africa gallery: select Nigeria.
- Nigeria map: click a tribe marker.
- Igbo village: collect yams and kola nuts, cook, and deliver to elders.
- Yoruba mission: collect drum sticks and complete a talking drum rhythm.
- Hausa mission: collect fabric and flags, then arrange the Durbar parade.
- Optional: Arochukwu storytelling puzzle near the story circle (Igbo).
- Festival ending: celebration and unity message.

## Controls
- Hub/Africa: drag to rotate, click markers to select.
- Village: WASD to move, click to look around, press E to interact.

## Audio
- Ambient sound and cue tones start after the first click or key press.

## Run Locally
- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build: `npm run build`
